# renovate

A Claude Code plugin that manages Renovate workflows including fixing failing PRs and installing Renovate configurations.

## Agent Skills

| Skill | Description |
|-------|-------------|
| [`renovate:fix-docker-openstack-patch`](skills/fix-docker-openstack-patch/SKILL.md) | Fix a failing patch in a Docker OpenStack image repository |

## Commands

| Command | Description |
|---------|-------------|
| [`/renovate:fix-docker-openstack-pr`](commands/fix-docker-openstack-pr.md) | Fix a single failing Renovate PR |
| [`/renovate:fix-docker-openstack-repo`](commands/fix-docker-openstack-repo.md) | Fix all failing Renovate PRs in a repository |
