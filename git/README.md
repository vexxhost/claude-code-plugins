# Git Plugin

A Claude Code plugin that streamlines Git workflows by generating standardized commit messages and pull requests.

## Agent Skills

| Skill | Description |
|-------|-------------|
| [commit-message](skills/commit-message/SKILL.md) | Generates commit messages with conventional commits and DCO compliance |
| [pull-request](skills/pull-request/SKILL.md) | Creates pull requests with proper formatting and reviewer suggestions |

## Commands

| Command | Description |
|---------|-------------|
| [`/commit-push-pr`](commands/commit-push-pr.md) | Create a commit, push, and open a pull request |
